1. 잔차(Residual)의 기본 개념
===============================================================================

[잔차의 정의]
잔차 = 실제값 - 예측값 (Residual = Observed - Predicted)

• 실제값: 실험에서 측정한 실제 데이터
• 예측값: 모델이 예측한 값 (그룹 평균)
• 잔차: 모델이 설명하지 못한 부분 (오차)

[현재 분석에서의 잔차]
• 잔차 개수: 18개 (각 실험 단위별)
• 잔차 평균: -0.000000 (이론적으로 0이어야 함)
• 잔차 표준편차: 2.4190mg
• 잔차 범위: -4.39 ~ 4.65mg

[잔차의 중요성]
• 모델의 적합성 진단
• 통계적 가정 검증
• 이상값 탐지
• 모델 개선 방향 제시

2. 잔차 vs 예측값 플롯 해석
===============================================================================

[플롯의 목적]
• 등분산성(Homoscedasticity) 가정 검증
• 모델의 선형성 확인
• 이상값 탐지
• 패턴 존재 여부 확인

[현재 결과 분석]
예측값: 3개의 고유값 (각 그룹 평균)
• 세정제 A: 17.12mg
• 세정제 B: 9.34mg  
• 세정제 C: 12.68mg

[이상적인 패턴]
✅ 좋은 패턴 (현재 우리 결과):
• 잔차가 0 주변에 무작위로 분포
• 예측값에 관계없이 일정한 분산
• 특별한 패턴이나 추세 없음
• 대부분의 점이 -3σ ~ +3σ 범위 내

[문제가 있는 패턴들]
❌ 나쁜 패턴들:
• 깔때기 모양: 분산이 증가하거나 감소
• 곡선 패턴: 비선형 관계 존재
• 군집화: 특정 영역에 점들이 몰림
• 극단값: 다른 점들과 크게 떨어진 점

[현재 결과 해석]
우리 데이터의 잔차 vs 예측값 플롯:
• 각 예측값(그룹 평균)에서 잔차가 고르게 분포
• 뚜렷한 패턴이나 추세 없음
• 분산이 예측값에 따라 크게 변하지 않음
→ 등분산성 가정 만족

3. Q-Q 플롯(Quantile-Quantile Plot) 해석
===============================================================================

[Q-Q 플롯의 목적]
• 잔차의 정규성 검증
• 이론적 정규분포와 실제 분포 비교
• 분포의 치우침이나 꼬리 특성 파악

[Q-Q 플롯 읽는 방법]
• X축: 이론적 정규분포의 분위수
• Y축: 실제 잔차의 분위수
• 대각선: 완벽한 정규분포일 때의 기준선

[이상적인 패턴]
✅ 정규성 만족 (현재 우리 결과):
• 점들이 대각선 근처에 위치
• 약간의 편차는 정상적
• 전체적으로 직선 패턴 유지

[문제가 있는 패턴들]
❌ 정규성 위반:
• S자 곡선: 분포가 치우침
• 아래쪽 꼬리가 길면: 왼쪽 치우침
• 위쪽 꼬리가 길면: 오른쪽 치우침
• 계단 모양: 이산적 분포

[현재 결과 해석]
Shapiro-Wilk 검정: p-값 = 0.7186 (> 0.05)
→ 잔차가 정규분포를 따름
→ Q-Q 플롯에서 점들이 대각선 근처에 위치

4. 잔차 정규성의 의미와 중요성
===============================================================================

[정규성이 중요한 이유]

1. 통계적 추론의 타당성
• F-검정의 정확성 보장
• 신뢰구간의 정확성 확보
• p-값의 신뢰성 보장

2. 모델 가정의 만족
• ANOVA의 기본 가정 중 하나
• 잔차 ~ N(0, σ²) 가정
• 통계적 검정의 유효성

3. 예측의 정확성
• 예측 구간의 정확성
• 미래 값 예측의 신뢰성
• 불확실성 정량화

[정규성이 위반되면?]

❌ 문제점들:
• F-검정 결과 신뢰성 저하
• Type I/II 오류율 증가
• 신뢰구간 부정확
• 예측 성능 저하

🔧 해결 방법들:
• 데이터 변환 (로그, 제곱근 등)
• 비모수 검정 사용
• 강건한 통계 방법 적용
• 표본 크기 증가
